# OTP_verification-Node.js-Twilio-

# OTP Verification (SMS using Twilio)

This is a simple Node.js + Express API that sends OTP to a phone number and verifies it using Twilio.

---

## Features

- Send OTP to phone number
- Verify OTP
- Store user in MongoDB
- Protected route after verification

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Twilio (SMS OTP)
